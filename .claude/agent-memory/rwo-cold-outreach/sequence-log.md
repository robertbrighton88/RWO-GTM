# Cold Outreach Sequence Log

All sequences generated by the Cold Outreach Agent. Read this BEFORE generating any new sequence to prevent duplicates (60-day window).

|---|---|---|---|---|---|---|

## Status Values
- **Generated** — sequence created, not yet imported to Outreach.io
- **Imported** — added to Outreach.io sequence
- **Active** — currently running in Outreach.io
- **Opened** — prospect opened at least one email
- **Replied** — prospect replied
- **Meeting booked** — call or meeting confirmed
- **No response** — sequence completed with no engagement
- **Bounced** — delivery failed
- **Paused** — sequence paused (contact left role, etc.)
